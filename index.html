Need to Understand a Little About How a Page Loads

This is Simplified But...

(1) Browser locates a website based on its URL

(2) Browser asks server for HTML, CSS, JavaScript, and images needed to make that website look good and be usable.

(3) Server sends it to the browser.

Enter the Performance Budget

How to Measure Performance Budget?

(1) Size of files needed

(2) Number of times the browser asks the server for files (each takes time)

(3) Time it takes for people to start seeing a page
